This is an R Shiny app project. My goal is to build an APP that can be used to evaluate a degree financially. 
The higher education in the States is more a business than a charity, so every young folk needs to make a wise choice before they commit to any degree. 
The factors discussed in this demo should not be the sole consideration in making this decision.

The product can be found here: https://pingliao.shinyapps.io/Quantify_Your_Degree/
